XBrainLab Homepage
=====================================

.. rst-class:: h5 text-center font-weight-light my-4

   XBrainLab is an open-source Python software for decoding EEG with deep learning.
   It incorporated commonly used data preprocessing methods along with a comprehensive
   analysis framework, including state-of-the-art neural network implementations, model
   training and evaluation pipeline and extracted feature visualization.

   XBrainLab provides an user-friendly graphic user interface (GUI) allowing interactive
   operations on custom data, as well as a history script generator for better analysis
   reproductibility and flexibility.
